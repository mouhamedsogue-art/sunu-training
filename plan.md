# SUNU Training Center - Professional HR & Training Platform

Build a modern, professional HR and training platform in French with three distinct interfaces: Candidate, Partner Enterprise, and Admin.

## Tech Stack
- React 19 + Vite
- Tailwind CSS 4
- Lucide React (Icons)
- Framer Motion (Animations)
- Sonner (Notifications)
- Shadcn UI (Components)

## UI/UX Chart
- Primary: #4169E1 (Royal Blue)
- Secondary: #FFFFFF, #F5F7FA (Light Gray), #1F2937 (Soft Black)
- Accent: #F59E0B (Light Orange)
- Typography: Poppins (Heading), Montserrat (Body) - via Google Fonts in index.html

## Structure
1. **Shared Layout**: Navigation system to switch between roles for demo purposes.
2. **Landing Page**: Fixed navbar, hero with AI illustration, stats, services.
3. **Candidate Interface**: 
   - Sidebar-based navigation.
   - Multi-step CV upload form (Personal, Academic, Skills, Upload).
   - Training catalog with cards and calendar.
4. **Partner Interface**:
   - Business dashboard.
   - Recruitment request form.
   - Tracking table for requests.
5. **Admin Interface**:
   - Advanced dashboard with RH analytics.
   - AI-powered CV analysis & filtering.
   - Candidate/Enterprise management.

## Implementation Details
- **CV Upload**: Realistic drag-and-drop simulation with progress bars.
- **AI Admin**: Visual indicators for AI match scores (e.g., "92% compatible").
- **Training**: Visual cards with "Présentiel" badges and registration buttons.
- **Responsiveness**: Fully responsive mobile-first design.
- **Language**: All content in French as requested.
